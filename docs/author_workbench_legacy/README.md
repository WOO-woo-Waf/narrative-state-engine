# 作者工作台旧阶段文档归档索引

本目录用于归档“对话主入口重构”之前的作者工作台设计、执行、联调和交付文档。旧文档仍然保留，用于查证历史决策，但后续新的前后端需求文档统一放到：

```text
docs/dialogue_first_workbench_rebuild/
```

## 目录分类

```text
01_concept_and_state
  统一小说状态、状态环境、图结构与对话概念的早期设计。

02_backend_plans
  后端状态环境、候选审计、批量审计、模型辅助审计等旧执行方案。

03_frontend_plans
  前端工作台、可读性修复、候选审计、模型辅助审计等旧执行方案。

04_integration_and_real_run
  前后端联调、真实 123 小说测试、问题记录、运行指南。

05_delivery_reports
  前端/后端阶段性交付报告和完成说明。
```

## 使用原则

1. 旧文档只作为历史依据。
2. 新开发不要继续在旧文档上追加需求。
3. 新需求以 `docs/dialogue_first_workbench_rebuild/` 为准。
4. 如果旧文档中的概念仍有价值，应复制并重写到新目录，而不是让执行窗口在多个目录之间来回找。
